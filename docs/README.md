## What is this?

`money-legos` provides you with the ABIs and addresses of popular Ethereum Mainnet DeFi protocols.

![autocomplete](https://github.com/studydefi/money-legos/blob/master/assets/legos-autocomplete.gif?raw=true)

## Protocols supported

- AAVE
- Compound
- Curve Finance
- DappSys
- DyDx
- ERC20
- MakerDAO
- OneSplit
- Uniswap v1
